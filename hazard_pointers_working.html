How Hazard Pointers Work:

  1. Protect: Thread sets hazard pointer before accessing a node
  2. Retire: Instead of freeing immediately, add node to retire list
  3. Scan: Check all hazard pointers - only free unprotected nodes
  4. Safe: No use-after-free bugs!
